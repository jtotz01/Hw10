Stock Choice: Solar Edge (SEDG)

Describe the API you selected and what it does.

The API I selected provides daily information of a stock. It shows the symbol, open price, high price, low price, close price, volume, latest trading day, previous close, change, and percentage change.

Describe the options you used for the API request.

For this API request, I used the "GET" request to process information about a specific stock when it comes from the data base.

Give two applications where this API could be helpful.

This API could be helpful for personal or professional purposes. Individual investors and professional investors refer to real time and historical stock data to make investing decisions. Stock APIs take complicated stock-related information and parse it in a way that is readable to individuals and accurate enough to provide the basis for investing decisions. For example, traders/investors might be creating a dashboard that allows them to visualize stock market data or they might be creating a trading algorithm that executes decisions based on real time stock data.
